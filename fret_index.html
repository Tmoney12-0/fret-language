Program with a guitar.

Fretlang is a guitar-driven programming language where chords become keywords, notes become letters and numbers, and guitar techniques become syntax actions. The goal is to make coding feel musical, physical, colorful, and performative.

Purpose and Philosophy

Fretlang turns guitar performance into code. Instead of treating music and programming as separate worlds, it merges them into a single playful system. Chords act like control words, notes act like symbols, and performance techniques become live syntax.

Implementation

Fretlang is built in three major layers so live guitar input can become executable programs.

1. Token Mapper

Musical events → tokens

Maps note, chord, and technique events into Fretlang tokens in real time.

2. textX Grammar

Tokens → parsed structure

Parses normalized .fret programs using a custom DSL grammar.

3. Python Runtime

AST → execution

Executes variables, loops, conditionals, and print statements through the interpreter.
